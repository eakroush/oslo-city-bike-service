# Retail Store Discount Application

Is an application to Calculate discount depending on Discount types.

### Prerequisites

- Java 11 and above
- Maven

### Installing

######Checkout the project from git

```
git clone path
```

######To run the application
```
mvn spring-boot:run
```

######To run tests

```
mvn clean verify
```


## API documentation
http://localhost:8181/city-bike/swagger-ui
